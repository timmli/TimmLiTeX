* latex-templates

Collection of LaTeX settings and templates 

** Features

- optimized for Emacs
- maintained in one =org-mode= file ([[file:SETUP.org][SETUP.org]])

** Included templates

- short abstract (tex)
- standard article (tex)
- standard book (tex)
- beamer slides (tex/org)
- langsci paper (tex/org)
- Hausarbeit & Essay (tex)
